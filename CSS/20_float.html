Uso de float en CSS

  • float:left/right mueve el elemento y deja fluir contenido alrededor.
  • Ideal para imágenes con texto alrededor.
  • Puede usarse para cajas, pero hoy en día se prefiere flex o grid.
  • Los elementos flotantes hacen colapsar el alto del contenedor: solución → overflow:hidden.
