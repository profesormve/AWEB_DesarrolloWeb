Comparativa entre unidades CSS: px, %, em, rem

Usando px

Este texto usa 20px. Es un valor fijo, independiente del contenedor o navegador.

Usando %

Este bloque ocupa 80% del ancho disponible. Es útil para diseños fluidos.

Usando em

El tamaño base es 20px.

Este texto usa 1.5em → 30px

Usando rem

El tamaño base global es 16px.

Este texto usa 2rem → 32px

Resumen comparativo de unidades

Unidad ¿Qué significa? ¿Cuándo usarla?
px Valor fijo en píxeles Cuando se necesita precisión absoluta (bordes, iconos)
% Relativo al tamaño del contenedor Diseños fluidos, adaptable a distintas pantallas
em Relativo al tamaño de fuente del padre Para escalar elementos internamente; cuidado con acumulación
rem Relativo al tamaño de fuente del html Para mantener consistencia y accesibilidad global

Consejos finales

  • Usa rem para textos, paddings y márgenes: mejora la legibilidad y accesibilidad.
  • Usa % para anchuras y layouts adaptables.
  • Usa em con cuidado en estructuras anidadas: se acumula.
  • Evita px para fuentes, salvo en casos específicos como iconos o bordes finos.
  • Establece una base clara en html con font-size (por ejemplo, 16px) y diseña usando rem.
