Uso las sombras en CSS (box-shadow)

  • Utiliza X e Y para definir la dirección de la sombra.
  • Usa blur para suavidad y spread expansión (tamaño que crece)
  • Usa rgba() para sombras profesionales con transparencia.
  • inset te permite crear sombras internas.
  • Combina sombras + hover para efectos dinámicos en UI.

Ejemplo 1 – Sombra con desplazamiento

Sombra desplazada: usa X e Y para mover la sombra.

box-shadow: 15px 10px 0px 0px #000;

Ejemplo 2 – Sombra en forma de borde

Sombra tipo borde: usa solo SPREAD para expandirse.

box-shadow: 0px 0px 0px 10px #cfcfcf;

Ejemplo 3 – Sombra difusa y suave

Sombra suave: mucho blur crea profundidad realista.

box-shadow: 0 0 40px 5px rgba(0,0,0,0.25);

Ejemplo 4 – Sombra interna (inset)

Sombra interna: se mete dentro del contenido.

box-shadow: inset 0 0 20px 5px rgba(0,0,0,0.3);

Ejemplo 5 – Sombra animada con hover

Pasa el ratón por encima: aparece la sombra con transición.

transition: box-shadow 0.5s;
box-shadow: 10px 12px 0px 0px #000; (al pasar el ratón)
